# KAVAN v6.0 — Security Policy

We take the security of KAVAN seriously. If you identify a security issue, please disclose it responsibly.

---

## 1. Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.0.x   | :white_check_mark: |
| < 6.0   | :x:                |

---

## 2. Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, email details of the vulnerability to:
`security@kavan.com`

Include the following details in your report:
- Description of the vulnerability and its potential impact.
- Step-by-step instructions or proof of concept code to reproduce.
- Any suggestions for mitigation or a patch.

We will acknowledge receipt of your report within 48 hours and work to provide a fix as quickly as possible.
